Introduction

Orchem contains some utility to convert different chemical structures. In ConvertMolucule.java class are located in Java package 'uk.ac.ebi.orchem.convert'. This class included:
  • SmilesToMolfile
  • MolfileToSmiles
You interact with these classes through PL/SQL packages that wrap the classes as Java stored procedures/functions. These packages will be explained in the following sections and some example queries will be given.

Smiles To Molfile

Package orchem_convert.SmilesToMolfile: The function SmilesToMolfile convert a Smiles to Molfile string:
  • SMILES (CLOB): a string SMILES string
  • RETURN (CLOB): a string MDL mol file converted
Examples
select orchem_convert.smilestomolfile('cc1cccc1') as mol from dual


Molfile To Smiles

Package orchem_convert.MolfileToSmiles: The function MolfileToSmiles convert a Molfile to Smiles string:
  • MOLFILE (CLOB): a string MDL mol file string
  • RETURN (CLOB): a string SMILES converted
Examples
select orchem_convert.molfiletosmiles(orchem_convert.smilestomolfile('cc1cccc1')) as smi from dual
